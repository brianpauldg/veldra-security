# 30-Day Post-Launch Roadmap (Days 15-44)

---

## Week 3 (Days 15-21): Optimize

| Task | Owner | Target |
|------|-------|--------|
| Review funnel data: quiz → book → pay → intake conversion rates | Founder | Identify biggest dropoff |
| A/B test homepage headline and CTA | Dev | Test running |
| Optimize Google Ads: pause low-performing keywords, increase bids on winners | Founder | CPC < $5, CPL < $30 |
| Scale ad spend to $75-100/day if CAC is viable | Founder | Budget adjusted |
| Process first lab results and prescriptions | Provider | 5-10 patients in treatment |
| Deploy missed-booking recovery emails (1hr, 24hr, 72hr) | Dev | Recovery sequences live |
| Deploy abandoned intake recovery emails | Dev | Intake reminders live |
| Request Trustpilot reviews from first satisfied patients | Ops | 3-5 reviews collected |
| Begin provider recruitment for 2nd provider | Founder | Job posted or network contacted |
| Sign first referral partner (gym, medspa, or wellness center) | Founder | 1 partner signed |

**Targets:** 10-15 active patients, $2,000-3,000 MRR

---

## Week 4 (Days 22-28): Expand

| Task | Owner | Target |
|------|-------|--------|

---

## Weeks 5-6 (Days 29-44): Stabilize & Scale

| Task | Owner | Target |
|------|-------|--------|
| Target 30-50 active patients | All | Patient count growing |
| Implement retention workflows (churn risk scoring, proactive outreach) | Dev | Retention system live |
| Launch referral program (patient referrals — "Refer a friend, get $50 credit") | Founder + Dev | Program live |
| Optimize email sequences based on open/click data | Dev | Sequences improved |
| Add Meta Ads (Instagram/Facebook) if Google Ads profitable | Founder | New channel tested |
| Build admin dashboard (basic user management, billing overview) | Dev | Admin features live |
| Implement vector search for AI assistant (if needed) | Dev | Search functional |
| Create "How We Compare" content (vs Hims, vs Marek, vs TRT Nation) | Founder + AI | Comparison pages published |
| Process first refills and track patient adherence | Provider + Ops | Refill workflow smooth |
| Weekly team standup established (founder + provider + ops) | Founder | Meeting cadence set |

**Targets:** 30-50 active patients, $6,000-10,000 MRR, <5% monthly churn
