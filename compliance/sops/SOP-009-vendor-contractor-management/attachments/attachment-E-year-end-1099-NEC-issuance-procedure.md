# SOP-009 Attachment E — Year-End 1099-NEC Issuance Procedure

**Parent Document:** [SOP-009 Vendor & Contractor Management (1099 Operations)](../SOP-009-vendor-contractor-management.md)
**Version:** 0.1

## Purpose

This attachment provides the step-by-step January workflow for 1099-NEC reconciliation, preparation, filing, and delivery. The bookkeeper executes this procedure annually. The MSO Owner reviews and approves before filing. The CPA confirms the filings during annual tax preparation.

---

## Key Deadlines

| Milestone | Deadline | Responsible |
| --- | --- | --- |
| Pull YTD payment totals | By January 10 | Bookkeeper |
| Identify reportable vendors and apply exemptions | By January 15 | Bookkeeper |
| Prepare 1099-NEC drafts | By January 20 | Bookkeeper |
| MSO Owner review and approval | By January 25 | MSO Owner |
| File 1099-NECs with IRS (e-file) | By January 31 | Bookkeeper |
| Deliver 1099-NEC copies to vendors | By January 31 | Bookkeeper |
| File California state copies (if applicable) | By January 31 | Bookkeeper |
| Reconciliation report archived | By February 15 | Bookkeeper |
| CPA review and confirmation | During annual tax prep (February–April) | CPA |

**The January 31 deadline is statutory.** Late filing results in IRS penalties that increase with the length of delay. The bookkeeper and MSO Owner must treat January 31 as a hard deadline with no extensions (barring IRS-announced deadline postponement).

## Procedure

### Week 1 (January 1–10): Data Pull and Identification

**Step 1 — Pull YTD payment totals.**
The bookkeeper generates the final YTD vendor payment report from the AP tool (QuickBooks or equivalent) for the preceding calendar year (January 1 through December 31). This is the authoritative data source for 1099 amounts.

**Step 2 — Reconcile against prior monthly reconciliations.**
The bookkeeper verifies that the YTD totals are consistent with the cumulative monthly reconciliation records. If any month's reconciliation was incomplete or had unresolved discrepancies, resolve them now before proceeding.

**Step 3 — Identify reportable vendors.**
The bookkeeper creates the 1099 issuance list by identifying all vendors with YTD nonemployee compensation of $600 or more.

**Step 4 — Apply corporate exemption.**
For each vendor on the 1099 list, the bookkeeper checks the W-9 entity type:

| W-9 Entity Type | 1099-NEC Required? | Exception |
| --- | --- | --- |
| Individual / Sole Proprietor | **Yes** (if ≥$600) | None |
| Single-Member LLC (disregarded entity) | **Yes** (if ≥$600) | None |
| Partnership / Multi-Member LLC taxed as partnership | **Yes** (if ≥$600) | None |
| S-Corporation | **Generally No** | **YES if medical/healthcare or attorney payments** |
| C-Corporation | **Generally No** | **YES if medical/healthcare or attorney payments** |
| LLC taxed as S-Corp | **Generally No** | **YES if medical/healthcare or attorney payments** |
| LLC taxed as C-Corp | **Generally No** | **YES if medical/healthcare or attorney payments** |

**Medical/legal exception:** Payments for medical and healthcare services (IRS 1099-NEC instructions, Box 1) and payments to attorneys (including attorney firms regardless of entity type) require 1099-NEC issuance regardless of the vendor's corporate status. The bookkeeper applies this exception: if the vendor provides medical or legal services, issue the 1099 regardless of entity type.

**When in doubt, issue the 1099.** Over-issuance (issuing a 1099 to a vendor that may be exempt) carries no penalty. Under-issuance (failing to issue a required 1099) carries IRS penalties.

---

### Week 2 (January 11–20): Preparation

**Step 5 — Verify W-9 information.**
For each vendor on the 1099 issuance list, the bookkeeper verifies:

| Item | Verification |
| --- | --- |
| Legal name on W-9 matches vendor record in AP tool | ☐ |
| TIN on W-9 is present and legible | ☐ |
| Address on W-9 is current (confirm with vendor if last updated >12 months ago) | ☐ |
| Entity type on W-9 matches classification analysis | ☐ |

If any W-9 is missing, expired, or has discrepancies, the bookkeeper requests an updated W-9 immediately per [Attachment C](attachment-C-w9-and-coi-collection-templates.md).

**Step 6 — Prepare 1099-NEC drafts.**
The bookkeeper generates draft 1099-NEC forms using the 1099 filing service (Track1099, QuickBooks 1099 module, or equivalent). Each draft includes:

- **Payer information:** Bloom Metabolics LLC, EIN, address
- **Payee information:** Vendor legal name, TIN, address (all from W-9)
- **Box 1 — Nonemployee compensation:** Total payments for the calendar year
- **State information (if applicable):** California state ID number and state tax withheld (typically $0 at v0.1)

**Step 7 — Cross-check totals.**
The bookkeeper cross-checks each draft 1099-NEC amount against the YTD vendor payment report and the vendor file payment log. All three sources must agree. If there is a discrepancy, investigate and resolve before proceeding.

---

### Week 3 (January 21–25): Review

**Step 8 — MSO Owner review.**
The MSO Owner reviews the complete 1099 issuance package:

- 1099 issuance list (which vendors, which amounts)
- Corporate exemption determinations (which vendors are exempt and why)
- Medical/legal exception applications
- Any vendors with discrepancies or missing W-9s
- Draft 1099-NEC forms

The MSO Owner signs off on the package. Any corrections are made before filing.

---

### Week 4 (January 26–31): Filing and Delivery

**Step 9 — File with IRS by January 31.**
The bookkeeper e-files all 1099-NECs with the IRS using an approved e-file service. E-filing is required if Bloom files 10 or more 1099 forms (per current IRS threshold — verify annually as the threshold has been lowered). Even below the threshold, e-filing is preferred for auditability and confirmation of receipt.

**Step 10 — Deliver to vendors by January 31.**
The bookkeeper delivers a copy of each 1099-NEC to the vendor:

| Delivery Method | When to Use | Requirements |
| --- | --- | --- |
| E-delivery | Preferred when vendor has consented | Written consent from vendor to receive 1099 electronically (via 1099 service consent workflow) |
| U.S. Mail | When vendor has not consented to e-delivery | Mailed to address on W-9; first-class mail; postmarked by January 31 |

**Step 11 — California state filing by January 31.**
California conforms to federal 1099 filing requirements. If Bloom paid California-resident contractors, the bookkeeper files California state 1099 copies. Filing methods:

- If using a 1099 filing service (Track1099, etc.), the service typically handles combined federal/state filing automatically
- If filing separately, file with the California Franchise Tax Board (FTB) by January 31
- Confirm state filing requirements with CPA annually — California requirements may differ from federal in edge cases

---

### February: Reconciliation and Archival

**Step 12 — Archive filing records.**
The bookkeeper retains the following in the 1099 archive folder:

- Copy of each 1099-NEC filed (payer copy)
- IRS e-file confirmation receipt
- State filing confirmation (if applicable)
- 1099 issuance list with amounts, exemptions, and exceptions
- YTD vendor payment report used as the source data
- MSO Owner sign-off

**Step 13 — CPA review.**
During annual tax preparation (typically February through April), the CPA reviews:

- 1099 issuance list for completeness
- Corporate exemption determinations
- Medical/legal exception applications
- Any vendors approaching but not meeting the $600 threshold
- State filing compliance

Any corrections identified during CPA review are filed using IRS Form 1099 corrected return procedures. Corrected 1099s are filed as soon as the error is identified — there is no deadline extension for corrections, but prompt filing reduces penalty exposure.

## Penalty Reference (Informational)

Late or incorrect 1099 filing can result in IRS penalties. The following is informational — consult the CPA for current penalty amounts, which are adjusted annually:

| Filing Delay | Approximate Penalty Per Form (2025 rates — verify annually) |
| --- | --- |
| Filed within 30 days of deadline | $60 per form |
| Filed more than 30 days late but by August 1 | $120 per form |
| Filed after August 1 or not filed | $310 per form |
| Intentional disregard | $630+ per form (no cap) |

**Small business exception:** Businesses with gross receipts ≤$5 million have lower maximum aggregate penalties. Consult CPA for current thresholds.

---

## Corrected 1099 Procedure

If an error is discovered after filing (wrong amount, wrong TIN, wrong name):

1. Bookkeeper identifies the error and notifies MSO Owner immediately
2. Bookkeeper prepares a corrected 1099-NEC using the 1099 filing service's correction workflow
3. Corrected 1099 is filed with IRS as soon as possible
4. Corrected copy is delivered to the vendor
5. CPA is notified of the correction
6. Correction is documented in the vendor file and the 1099 archive
