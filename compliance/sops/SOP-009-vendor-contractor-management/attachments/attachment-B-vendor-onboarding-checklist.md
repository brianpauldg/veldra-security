# SOP-009 Attachment B — Vendor Onboarding Checklist

**Parent Document:** [SOP-009 Vendor & Contractor Management (1099 Operations)](../SOP-009-vendor-contractor-management.md)
**Version:** 0.1
**Last Updated:** 2026-05-11

---

## Purpose

This checklist is completed for every new vendor before the first payment is issued. It ensures that all required documentation is collected, verified, and filed before the vendor is cleared for payment. No vendor may receive payment until all applicable items are complete and the MSO Owner has signed the clearance certification at the bottom of this checklist.

---

## Instructions

1. The MSO Owner or bookkeeper initiates this checklist when a new vendor engagement is approved.
2. Complete each item in order. Mark each item as complete with the date completed and the initials of the person who completed it.
3. Items marked "If applicable" are required only when the condition described applies to this vendor. If not applicable, mark "N/A" with a brief reason.
4. File the completed checklist in the vendor file at `compliance/vendor-files/[vendor-name]/`.

## Onboarding Checklist

| # | Item | Required? | Source | Format | Verification | Date | Initials |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 1 | **Classification analysis completed** ([Attachment A](attachment-A-classification-decision-tree.md)) | Required for all vendors | MSO Owner runs the classification decision tree | Completed worksheet | Signed by MSO Owner; all three ABC prongs documented; classification decision rendered | | |
| 2 | **Scope of work / engagement letter / contractor agreement executed** | Required for all vendors | Drafted by MSO Owner using standard IC agreement template; legal review for non-standard terms or >$5,000 annualized | Executed PDF (both parties signed) | Filed in vendor file; terms reviewed by MSO Owner | | |
| 3 | **IRS Form W-9 collected** | Required for all vendors — no exceptions | Vendor completes IRS Form W-9 | Signed PDF | TIN present and legible; entity type checked; address complete; signature and date present. TIN match against IRS TIN matching service if registered | | |
| 4 | **Certificate of Insurance (COI) collected** | If applicable — required for: on-site work, professional services with E&O exposure, vendors creating third-party liability risk | Vendor's insurance carrier issues COI | PDF | Reviewed by MSO Owner: coverage type matches scope, minimum limits met, Bloom named as additional insured or certificate holder where required, policy is current | | |
| 5 | **HIPAA Business Associate Agreement (BAA) executed** | If applicable — required for any vendor with access to PHI | Drafted by MSO Owner using standard BAA template; vendor signs | Executed PDF (both parties signed) | Filed in vendor file; BAA covers all anticipated PHI access | | |
| 6 | **Bank / payment information collected** | Required for all vendors | Vendor provides via secure form or secure email | Encrypted or stored in AP tool only | Stored in accounts payable tool (QuickBooks, Bill.com); NOT stored in the vendor file folder as a standalone document | | |
| 7 | **Vendor entered in accounts payable tool** | Required for all vendors | MSO Owner or bookkeeper creates vendor record | Database entry in QuickBooks, Bill.com, or current AP tool | Legal name matches W-9; TIN entered; payment method configured; address matches W-9 | | |
| 8 | **Vendor file folder created** | Required for all vendors | MSO Owner or bookkeeper creates folder | Folder at `compliance/vendor-files/[vendor-name]/` | Folder exists; classification analysis, agreement, W-9, and any applicable COI/BAA are filed | | |
