## How Each Channel Works

### Local SEO → Discovery
Men search high-intent keywords: "online trt clinic" (8,100/mo), "trt telehealth" (3,600/mo), "glp-1 weight loss online" (2,900/mo). Bloom Metabolics pages rank → organic clicks → site visits. Zero cost, compounds over time.

### Directory Listings → Trust + Backlinks
NAP consistency across 25+ directories builds domain authority. Healthgrades, Zocdoc, Vitals create referral traffic from health-seekers. When prospects research "Bloom Metabolics reviews," they find verified listings everywhere. Trust signal.

### Website → Capture
Quiz funnels (email gated before results) convert 30-40% of takers. Scroll popups add another 2-5% capture rate. Inline forms on treatment pages catch engaged readers. Exit intent catches bouncing visitors. No visitor leaves without at least seeing a capture opportunity.

### Nurture → Booking
7-email sequence over 21 days: deliver value (guide), educate (hormones), social proof (patients), demystify (process), crush objections (safety/cost), create urgency (90-day question), graceful close. Target: 8-12% sequence-to-booking conversion.

### Agent System → Continuous Improvement
PM Agent orchestrates daily. Research Agent finds opportunities. Content Agent creates. QA Agent reviews. Analytics Agent measures. Weekly optimization cycle means the system gets better every week.

## Agent Workflows

### SEO Research Agent
- **Inputs:** Target keyword clusters, competitor URLs (hims.com, ro.com, marekhealth.com, petermd.com), Search Console data
- **Outputs:** Keyword maps (keyword, volume, difficulty, intent, target page), content briefs, weekly ranking report, gap analysis
- **Completion Criteria:** Keyword map covers all 4 clusters, no keyword cannibalization, all keywords have assigned target pages
- **QA Requirements:** Volume data sourced, intent classification verified, no duplicate keyword-page assignments

### SEO Content Agent
- **Inputs:** Content briefs from Research Agent, keyword targets, brand voice (.agents/product-marketing-context.md), compliance rules (CLAUDE.md)
- **Outputs:** Complete page content, title tag (<60 chars), meta description (<155 chars), FAQ blocks (5+ questions), internal link list
- **Completion Criteria:** Content meets word count target, all metadata present, FAQ schema-ready, compliance language used
- **QA Requirements:** No absolute health claims, disclaimers present, readability Grade 8, keyword density 1-2%

### SEO Implementation Agent
- **Inputs:** QA-approved content, page template (Next.js components), design system (Section, Card, Button)
- **Outputs:** Deployed page at correct route, metadata in place, schema markup, internal links, mobile responsive
- **Completion Criteria:** Page live on Vercel, indexed in Search Console, schema validates, PageSpeed >90
- **QA Requirements:** All links work, forms submit, mobile renders correctly, no console errors

### QA Agent
- **Inputs:** Any deliverable from any agent
- **Outputs:** Pass/fail with specific line-item feedback, compliance audit, brand consistency check
- **Completion Criteria:** All checklist items reviewed, verdict with reasoning
- **QA Requirements:** Compliance check (no absolute claims), brand check (matches product-marketing-context.md), technical check (links, forms, rendering)

### Reporting Agent
- **Inputs:** Search Console data, GA4 data, GHL pipeline data
- **Outputs:** Weekly SEO report with traffic, keyword rankings, conversion metrics, WoW comparisons
- **Completion Criteria:** All data sources accessed, calculations verified, report formatted and distributed
- **QA Requirements:** 3 metrics spot-checked for accuracy, trends correctly identified

## Directory Agent Ownership

### Directory Research Agent
- **Mission:** Find new directory opportunities, assess ROI
- **Inputs:** Industry keywords, competitor directory presence
- **Outputs:** New directory recommendations with priority and benefit
- **Triggers:** Monthly scan, on competitor directory spotted

### Listing Submission Prep Agent
- **Mission:** Adapt submission kit to each platform's requirements
- **Inputs:** Submission kit, platform field requirements
- **Outputs:** Platform-specific profile content ready for submission
- **Triggers:** New directory assigned for submission

### Listing Submission Agent
- **Mission:** Actually submit listings
- **Inputs:** Prepared content, credentials
- **Outputs:** Confirmation, verification method noted
- **Triggers:** Prep complete
- **Failure:** Rejected → note reason, fix, retry once → escalate

### Verification Agent
- **Mission:** Confirm listings are live, accurate, and claimed
- **Inputs:** Tracker with submission dates
- **Outputs:** Status updates, inconsistency flags
- **Triggers:** 7 days post-submission, monthly audit
- **Failure:** Needs phone/mail verification → escalate to human

### Tracker Update Agent
- **Mission:** Keep submission tracker current
- **Inputs:** Status changes from all agents
- **Outputs:** Updated tracker, summary stats
- **Triggers:** Any status change
