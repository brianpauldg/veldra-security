# Bloom Metabolics — Clinical OS + Marketing

## Project
Bloom Metabolics is a premium telehealth platform for TRT, GLP-1 weight loss, and peptide therapy. Next.js 14 (App Router), TypeScript, Tailwind CSS v4, Framer Motion.

## Compliance Rules
- Never make absolute health claims — use "may support," "commonly report," "individual results vary"
- AI is operational support only — never position as autonomous medical care
- Include disclaimers on testimonials and outcome claims
- HIPAA-compliant language in all patient-facing copy
